## CRUD-приложение для склада товаров

- Приложение создано при помощи технологий, указанных в тз.
- Подключен Swagger, доступен по http://localhost:8080/swagger-ui/
- В качестве первичного ключа используется UUID
- Использован Lombok
- Реализована валидация полей на null
- Настроены разные профили для локального старта приложения (в разных профилях разные порты)
- В проекте есть коллекция Postman с запросами и тестами (Перед запуском необходимо указать существующие id в запросах)
- Создан глобальный обработчик исключений (GlobalExceptionHandler.java)
- В коде есть комментарии Javadocs
- Написаны dockerfile и docker-compose

Для запуска приложения достаточно ввести команду:
'docker-compose up'

Решение и инструкция находятся в ветке develop
В ветке main находится только пустой README.md
